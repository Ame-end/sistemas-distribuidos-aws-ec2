## Descripción del Proyecto

Este proyecto consiste en el despliegue de una página web estática en una instancia EC2 de Amazon Web Services, utilizando exclusivamente servicios de la **capa gratuita (Free Tier)** de AWS.

La página web tiene temática espacial con animaciones CSS, estrellas generadas dinámicamente con JavaScript, un planeta animado y una nave espacial en movimiento.

---

## Tecnologías y Servicios Utilizados

| Tecnología / Servicio | Descripción |
|---|---|
| **Amazon EC2** | Servidor virtual en la nube (instancia t3.micro) |
| **Ubuntu Server 26.04 LTS** | Sistema operativo del servidor |
| **Apache HTTP Server 2.4** | Servidor web que sirve la página |
| **AWS Security Group** | Firewall virtual — puertos 22, 80 y 443 |
| **AWS VPC** | Red privada virtual predeterminada |
| **HTML5 + CSS3** | Estructura y estilos de la página web |
| **JavaScript** | Generación dinámica de estrellas animadas |
| **SSH (OpenSSH)** | Protocolo de acceso remoto al servidor |
| **Git + GitHub** | Control de versiones del código fuente |
